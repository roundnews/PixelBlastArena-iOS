// Pixel Blast Arena

// A retro, bomb‑blasting arcade game for iPhone and iPad. Navigate a grid, outsmart monsters, place bombs with chain reactions, collect power‑ups, and race to the portal.

// - App Store: https://apps.apple.com/app/id6757359436
// - GitHub: https://github.com/roundnews/PixelBlastArena-iOS
// - Author: Mazilu Mircea

// Description
// Pixel Blast Arena blends classic arcade vibes with modern SwiftUI + SpriteKit. Each level ramps up the challenge: more monsters, faster movement, evolving visuals, and smarter survival decisions. Clear enemies to activate the portal, advance levels, and chase the win.

// Features
// - Core gameplay
//   - Tight, grid‑based movement with an on‑screen D‑pad
//   - Place bombs to destroy crates and enemies; explosions chain when bombs overlap
//   - Power‑ups that enhance speed, bombs, pass‑through, and more
//   - Portal appears when all enemies are defeated — step in to advance
// - Progression and visuals
//   - Increasing monster count and speed per level
//   - Monster colors shift per level for immediate visual variety
//   - Crate/brick visuals gradually blend to grayscale by level 5
//   - “Home is close!” message at level 10; you win at level 15
//   - Level‑based power‑up timers: 15s at level 1, +30s per level up to level 5
//   - From level 5 onward: invincibility + pass‑through (announced in‑game)
// - Controls and UX
//   - D‑pad for movement; bomb button on the lower‑right
//   - Double‑tap anywhere on the playfield to place a bomb
//   - Once double‑tap is used, the bomb button can be hidden for a cleaner UI
//   - D‑pad can be repositioned: long‑press to arm, then tap to place it elsewhere
//   - Lightweight, center‑screen announcements for power‑ups and hints
// - Audio
//   - Background music with intelligent ducking during sound effects
//   - Layered bomb SFX for stronger presence
// - Polish
//   - Smooth camera follow
//   - Subtle animations for movement, explosions, and portals

// Controls
// - Move: On‑screen D‑pad
// - Place bomb: Bomb button (bottom‑right) or double‑tap anywhere on the game area
// - Pause/Resume: Top‑right button
